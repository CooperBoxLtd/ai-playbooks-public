# Contributing

Thank you for improving the CooperBox AI Coding Playbooks.

This repository is for vendor-neutral AI engineering knowledge that can help coding agents and AI-assisted software teams work more safely and consistently.

## Good Contributions

Preferred contributions:
- new principles that remain true across projects or stacks
- reusable implementation patterns
- assessable controls
- concrete evidence signals
- clearer playbook prose
- typo, structure, and consistency fixes

## Keep Out

Do not contribute:
- confidential material
- customer-specific details
- secrets, credentials, tokens, or private data
- product-specific strategy
- marketing copy
- certification or compliance claims
- vendor-specific recommendations unless they are generalized

## Contribution Standard

Before opening a pull request:
- map new guidance to an existing capability where possible
- avoid adding new top-level capability categories
- keep claims actionable and evidence-oriented
- prefer concise language over long essays
- update related registry, pattern, control, or evidence records when relevant
